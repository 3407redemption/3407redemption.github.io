Robot Progress: Push Back V1

Chassis Complete

Date:

We have completed the chassis that will be the base for the rest of our robot. It uses a 36:48 gear ratio which means an RPM of 450.

Intake Complete

Date:

The intake is now done, and we are able to score on all 4 goals. It uses rollers made from sprockets and rubber bands except for at the top where it uses flexwheels.

Hoarder Complete

Date:

We have now also completed the hoarder which we will use to store up to 15 blocks. The structure is made of standoffs and is covered with mesh so the blocks don't fall out. There is also an indexer that allows for us to intake and outtake from the hoarder.

Scraper Complete

Date:

We have completed the scraper which we will use to empty out loaders for the blocks. It uses polycarbonate to create a wedge that makes the blocks come out.
